# Сравнение метрик качества регрессии: MAE vs MSE

Этот проект представляет собой исследование трех моделей машинного обучения на датасете California Housing. Главная цель — сравнить метрики MAE (Mean Absolute Error) и MSE (Mean Squared Error), проанализировать их чувствительность к выбросам и понять, в каких случаях какую метрику лучше использовать.

## Задачи проекта

- Обучить три модели регрессии: Linear Regression, Random Forest, Gradient Boosting
- Сравнить их качество с помощью метрик MAE, MSE, RMSE
- Визуализировать результаты и проанализировать влияние выбросов
- Сделать выводы о выборе метрики в зависимости от бизнес-задачи

## Используемые технологии

- **Python 3.9+** - основной язык программирования
- **Pandas, NumPy** - обработка и анализ данных
- **Scikit-learn** - построение моделей машинного обучения
- **Matplotlib, Seaborn** — визуализация результатов
